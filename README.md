# 小猪基金：基金交易平台
___

## 项目介绍
本项目旨在开发一款能够提供多种功能的基金产品交易平台，针对不同的用户，提供不同的个性化的服务，从而达到相关核心目标：
1.提供清爽的操作界面以及简单的操作方法,以达到用户入手即会操作；
2.能够清楚地向不同用户提供相对应的基金，及提供个性化推荐选择功能；
3.增强用户对基金理财的理解，让用户在理财的同时学到更多理财知识；
